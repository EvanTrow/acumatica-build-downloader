# Acumatica-Build-Downloader

Downloads and extracts Acumatica MSI - **_Only works with Windows_**

# Requirements

-   [NodeJS](https://nodejs.org/en/download/)
-   PowerShell

# Install

Installs dependencies.

```shell
npm install
```

# Use

To start the script just run the following command

```shell
npm install
```
